If the dashboard does not load

The HTML UI is served by the Spring Boot app in this repository’s java/ module. From a checkout of this repo, start the server: 

cd java
./mvnw spring-boot:run

Windows (PowerShell), from the repository root:

cd java
.\mvnw.cmd spring-boot:run

Then open http://localhost:8080/ (default port; see application.yml).

You can open this file directly from disk (the path to this run-dashboard.html on your machine) without starting Java — it is the same checklist as the main dashboard’s “Run locally” box (absolute paths there).
